PANDUAN PENGGUNAAN

Cara menggunakan Cash-in

Ikuti langkah berikut untuk memulai menggunakan aplikasi Cash-in di perangkat Android Anda.

01

Download & Install APK

Unduh file APK melalui tombol download di bawah. Aktifkan opsi Instal dari sumber tidak dikenal di Pengaturan Android Anda, lalu buka file APK yang telah diunduh untuk memulai instalasi.

02

Buka Aplikasi & Daftar Akun

Buka aplikasi Cash-in. Pada halaman login, ketuk DAFTAR AKUN BARU. Isi formulir dengan User ID unik, password minimal 4 karakter, dan email (opsional). Ketuk SIMPAN & DAFTAR untuk membuat akun.

03

Login ke Akun Anda

Masukkan User ID dan password yang telah dibuat, lalu ketuk MASUK. Sesi login akan tersimpan otomatis sehingga Anda tidak perlu login ulang setiap membuka aplikasi. Kredensial default: admin / admin.

04

Input Data Keuangan

Ketuk tab INPUT DATA di bagian bawah. Pilih tipe transaksi (Pemasukan atau Pengeluaran), isi keterangan, masukkan nominal dalam Rupiah, dan pilih tanggal transaksi. Ketuk SIMPAN DATA untuk menyimpan.

05

Pantau di Dashboard

Buka tab DASHBOARD untuk melihat ringkasan dan grafik keuangan. Gunakan filter tanggal untuk melihat data pada periode tertentu. Grafik menampilkan tren pemasukan (hijau) dan pengeluaran (merah).
